# GetMedia

**📋 Description**

A powerful portable media downloader for Windows built with Batch script and yt-dlp.  
GetMedia provides a clean interactive terminal interface for downloading videos, audio, subtitles, thumbnails, and playlists from hundreds of supported websites while remaining lightweight, private, and easy to use.

---

**🎯 Key Features**

- **Portable Design:** No installation required — simply download and run  
- **Interactive Terminal UI:** Easy-to-use menu system for navigation and downloads  
- **Smart Playlist Handling:**  
  - Single-item download mode enabled by default  
  - Optional full playlist downloading  
  - Prevents accidental massive downloads  
- **Multiple Download Types:**  
  - Video downloads  
  - Audio extraction  
  - Subtitle downloads  
  - Thumbnail downloads  
- **CTRL+C Protection:** Safely cancels downloads and returns to main menu instead of terminating the script  
- **Performance Tuning:**  
  - Concurrent fragment downloading  
  - Retry handling  
  - Download speed limiting  
- **FFmpeg Integration:** Automatic merging and conversion support  
- **Privacy-Focused:** No tracking, telemetry, ads, or cloud dependency  
- **Wide Website Support:** Powered by yt-dlp for compatibility with hundreds of platforms  
- **Lightweight:** Simple Batch implementation with minimal resource usage
